# QR Code Generator

Gerador de QR Code interativo desenvolvido com HTML, CSS e JavaScript pelo site Frontend-Mentor.

##  Funcionalidades -

- Geração dinâmica de QR Code
- Validação de input
- Conversão automática de URL

##  Tecnologias -

- HTML
- CSS
- JavaScript
